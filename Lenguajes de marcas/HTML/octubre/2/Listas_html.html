Los atributos son elementos que me permiten modificar o configurar las etiquetas de HTML como yo desee. Se colocan dentro de la etiqueta de apertura, y tras ellos siempre sigue un igual (=) con unas comillas dobles (" ") y dentro de dichas comillas el valor que quiera darle a dicho atributo. Lo vamos a comprender muy bien con este ejemplo.
