Una Página Web es un documento electrónico que forma parte de la WWW (World Wide Web) generalmente construido en el lenguaje HTML (Hyper Text Markup Language o Lenguaje de Marcado de Hipertexto) ó en XHTML (eXtensible Hyper Text Markup Language o Lenguaje de Marcado de Hipertexto Extensible). La forma más básica de añadir un texto a una página web es usando la etiqueta. Aunque en las páginas anteriores te contamos sobre esta etiqueta, recordemos un poco: el elemento párrafo es uno de más usados en HTML para crear y construir los sitios web que visitas a diario.
